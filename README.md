# Decision Assembly Platform

A comprehensive platform designed to streamline decision-making processes through advanced analytics, automation, and integrated tools.

## 🚀 Overview

The **Decision Assembly Platform** (DAP) provides a suite of tools for business intelligence, data crawling, and specialized application frameworks.

## 📁 Repository Structure

-   **backend/**: Core services including crawlers, database initializers, and main logic.
-   **banking-app/**: Specialized application for financial decision support.
-   **demo-store/**: Demonstration interface for e-commerce integration.
-   **docs/**: Technical documentation and architecture overviews.
-   **ecommerce-website/**: Sample e-commerce frontend.
-   **sdk/**: Software Development Kit for extending the platform.
-   **scripts/**: Utility scripts for deployment and maintenance.
-   **test-sites/**: Sandbox environments for testing crawl and analysis logic.

## 🛠️ Setup & Installation

1.  **Backend Setup**:
    ```bash
    pip install -r requirements.txt
    python backend/main.py
    ```

2.  **Frontend/Apps**:
    Navigate to the specific app directory (e.g., `banking-app`) and run:
    ```bash
    npm install
    npm run dev
    ```
